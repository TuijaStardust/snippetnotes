## Plugin header
- Location: primary plugin file
- The only required lind in plugin header is the PLUGIN NAME
- The rest of the info is optional BUT RECOMMENDED

'''
<?php
/*
Plugin name:
Plugin URI:
Description: 
Version: 
Author:
Author URI:
Text Domain:
License:
*/
'''
